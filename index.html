My pagination doesn't work?

Make sure you use PHP version 7.3 or 7.4. If you are using cPanel, there should be a feature called "MultiPHP Manager" or so, select your site and set the version to "alt-php-73". This will work 99% of the time.

Mononoke is an Anime-Content-Management-System fully developed in PHP. It has a lot of features and can power big communities. It is Open-source and free, available to everyone.
